n 블로그 지침

* 선행조건
- 모든 요소는 개인 블로그에도 적용된다.
- 1팀 기준 다음 블로그들이 존재한다. (이하 1팀 블로그라고 한다.)
	=> 김팀장, 예랑이, 염소별자리, 아이엔, 굿데이, 수건, 스토리투유, 러닝스터디, 후기블



* 순서
- 웨일 브라우저에서 모바일 창을 킨다.
	-> 모바일 창으로 1팀 블로그 중 해당 로테이션의 블로그에 들어간다
		-> 블덱스로 들어가 검색하는데, 기존 키워드 중 검색량이 높은 것과, 한달 이내 중복을 회피한 키워드를 선택한다.
			-> 네이버에 조건을 만족한 키워드를 브라우저창에 쳐본다.
				-> 키워드 작성글이 블로그 순위에 들어갔다면, 1팀 블로그보다 아래의 블로그에 들어간다.
					-> 해당 블로그에서 대문 작성글 키워드부터 블덱스에 검색량을 조사한다.
						-> 블덱스에 다시 키워드 검색량을 보고, 만족한다면 해당 글을 키워드로 삼아 발행한다.
							-> 그러면 다른 블로그의 자리를 차지할 수 있다.
